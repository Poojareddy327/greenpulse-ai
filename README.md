# 🌍 GreenPulse AI - Carbon Footprint Awareness Platform

An AI-powered **Carbon Footprint Awareness Platform** that helps users measure, understand, and reduce their CO₂ emissions through interactive analytics, personalized AI recommendations, offset marketplace, and gamified challenges.

## ✨ Core Carbon Features

- 🏠 **Modern Landing Page** - Carbon footprint awareness messaging
- 📊 **Carbon Footprint Calculator** - Calculate personal CO₂ emissions from transport, energy, diet, and more
- 📈 **Carbon Analytics Dashboard** - Track emissions trends, category breakdown, and progress over time
- 🌍 **Global Comparison Tool** - Compare your footprint with world averages and climate targets
- 🛒 **Carbon Offset Marketplace** - Purchase verified carbon credits to offset your emissions
- 🤖 **AI Carbon Advisor** - Get personalized recommendations to reduce your carbon footprint
- 🎯 **Sustainability Challenges** - Complete eco-friendly challenges to reduce emissions
- 📚 **Climate Learning Hub** - Educational content about carbon footprint and climate change

### AI Integration
- Gemini API / OpenAI API

### Deployment
- Vercel (Frontend)
- Render (Backend)
- Neon PostgreSQL

## 🚀 Getting Started

### Prerequisites
- Node.js 18+
- Python 3.9+
- PostgreSQL

### Frontend Setup
```bash
cd frontend
npm install
npm run dev
```

### Backend Setup
```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```
